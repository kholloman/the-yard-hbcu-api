# the-yard-hbcu-api


## Prerequisites
* JDK 17+
* Maven 3.8+

## How to Run the Application
1. Open the terminal.
2. Run the command: `mvn spring-boot:run`
3. The API will be available at port 8080.

## How to Run Tests
1. Open the terminal.
2. Run the command: `mvn test`

## Sample API Requests
* **Get All:** `GET /api/opportunities`
* **Filter by Type:** `GET /api/opportunities?type=Scholarship`
* **Search by Keyword:** `GET /api/opportunities?q=STEM`
